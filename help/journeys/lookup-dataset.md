# 建立查詢資料集

資料集查詢可讓 Adobe Journey Optimizer 在歷程執行階段，從 Adobe Experience Platform 資料集擷取參考或交易型資料，而不需要將該資料儲存在即時客戶輪廓上。

在此使用案例中，當使用者提交投資偏好表單 (股票、債券或 CD) 時，歷程會動態查詢參考資料集中對應的投資詳細資料，並使用該資訊來將電子郵件個人化。
