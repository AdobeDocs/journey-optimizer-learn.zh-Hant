# 測量效能並最佳化

存取報表、分析效能，以及調整策略以取得更好的結果：

1. 追蹤關鍵量度：傳送、開啟、點按和轉換。
2. 分析對象績效以瞭解參與模式。
3. 根據資料深入分析精簡訊息時間、內容和目標。
