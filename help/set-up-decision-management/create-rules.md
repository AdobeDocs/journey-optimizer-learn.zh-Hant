# 建立規則

了解如何在 [!DNL Offer Decisioning] 中建立規則。規則或「決策規則」是個人化優惠方案的必要建置區塊元件之一。 規則是以 Platform 即時客戶設定檔中的事件和屬性所建立，可形成優惠的資格限制。例如，規則可用來限制給生活在下雪地區的客戶提供雪鏟的優惠方案。
