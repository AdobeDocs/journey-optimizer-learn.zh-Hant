# 建立規則

了解如何在[!DNL Offer Decisioning]中建立規則。 規則或「決策規則」是個人化優惠方案的必要建置區塊元件之一。 規則是使用Platform即時客戶設定檔中的事件和屬性，以及優惠方案的資格限制所建立。 例如，規則可用來限制雪鏟的選件給生活在下雪地區的客戶。
