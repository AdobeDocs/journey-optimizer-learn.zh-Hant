# 建立個人化優惠

了解如何在[!DNL Offer Decisioning]中建立個人化優惠方案。 個人化優惠方案具有與其相關聯的適用性規則，可協助您只向相關客戶顯示。 例如，您可以將雪鏟的優惠限制在雪下的地區，而且只能在冬季進行。
