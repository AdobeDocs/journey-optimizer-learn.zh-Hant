# 建立個人化優惠方案

了解如何在 [!DNL Offer Decisioning] 中建立個人化優惠。個人化優惠具有與其相關的適用性規則，可協助您僅向相關客戶顯示。 例如，您可以將雪鏟的優惠方案限制在下雪地區，而且只能在冬季進行。
