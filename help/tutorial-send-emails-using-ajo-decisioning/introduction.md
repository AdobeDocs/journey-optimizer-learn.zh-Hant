# 使用Decisioning個人化電子郵件

在本教學課程中，我們將探索使用Adobe Journey Optimizer Decisioning傳送個人化電子郵件優惠方案的實際使用案例。 情境很簡單 — 根據客戶的設定檔屬性（例如年收入和忠誠度狀態），傳送目標電子郵件優惠給客戶。
在本教學課程中，我們將逐步進行 — 從Adobe Experience Platform中建立設定檔，到設定電子郵件通道，最後透過決策建立並執行自動選取最相關優惠方案的行銷活動。

## 本教學課程的先決條件

* 存取Experience Platform

* 基本瞭解Experience Platform概念（設定檔、對象、資料集）

* 熟悉Journey Optimizer
