# 擷取設定檔並建立對象

在這段影片中，瞭解如何建立設定檔結構、從CSV檔案擷取客戶資料，以及在Adobe Experience Platform中使用設定檔屬性來建立對象。

您會瞭解如何定義重要欄位，例如名字、電子郵件、忠誠度狀態和年收入，上傳真實的客戶資料，並使用該資料建立高價值客戶的對象。
