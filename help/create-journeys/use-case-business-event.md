# 使用案例：商業事件

了解如何在 Journey Optimizer 使用商業事件。商業事件可讓品牌根據非人員相關事件觸發歷程，例如當新聞出版物刊出新內容時通知訂閱者，或在體育賽事期間傳送通訊。了解如何將業商業事件的資料用於歷程中的個人化傳訊功能。
