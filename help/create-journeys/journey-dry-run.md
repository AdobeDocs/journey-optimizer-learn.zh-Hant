# 使用歷程試運行驗證歷程邏輯

本教學課程介紹 Adobe Journey Optimizer 中的歷程試運行功能，此功能可讓行銷人員測試歷程邏輯，而不需傳送真正的通訊。您將瞭解如何在試運行模式中啟動歷程，設定如停用等待時間和外部資料來源等選項，並解譯設定檔流量量度以驗證客群細分和條件式邏輯。瞭解如何識別和修正問題 (例如條件節點中的拼寫錯誤)，以及如何在啟動歷程即時版之前存取和匯出試運行報告以供進一步分析。
