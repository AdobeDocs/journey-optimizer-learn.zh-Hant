# 建立行銷活動

在此步驟中，您將在Adobe Journey Optimizer中建立行銷活動，將排程的網頁推播通知傳送給已選擇加入的使用者。 行銷活動會鎖定符合資格的對象，並在預先定義的時間傳送訊息，以啟用計畫和對象為基礎的參與。

* 登入Journey Optimizer
* 導覽至Journey Management |行銷活動|建立行銷活動

## 指定Campaign設定


指定行銷活動名稱

![行銷活動名稱](assets/campign-push-notification.png)

## 將動作與行銷活動建立關聯

與本教學課程先前建立的推播通道設定建立關聯

![行銷活動動作](assets/campign-push-notification-action.png)

## 將對象與行銷活動建立關聯

將對象`AudienceForPush`與行銷活動建立關聯

![行銷活動 — 對象](assets/campign-push-notification-audience.png)

## 建立推播通知的內容

建立測試推播通知的基本推播內容。 指定訊息的標題和內文，如下所示

![推播通知內容](assets/campign-push-notification-content.png)

## 安排行銷活動

視需要排程行銷活動

![排程 — 行銷活動](assets/campign-push-notification-schedule.png)

最後，請確定您已啟動行銷活動。

## 測試行銷活動

若要測試行銷活動，請先在提示時選擇加入[，在](http://localhost:3000)網頁上啟用通知。 在您選擇加入後，請等待行銷活動在其排程時間執行。 行銷活動執行時，您應會在瀏覽器中收到推播通知。
