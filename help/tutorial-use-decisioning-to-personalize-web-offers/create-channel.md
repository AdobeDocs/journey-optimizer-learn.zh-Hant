# 建立程式碼型體驗管道

Adobe Journey Optimizer (AJO) Decisioning中的程式碼型體驗是一項設定，可讓您使用使用者端JavaScript直接將個人化優惠傳送至網頁。 如此一來，開發人員就能完全控制使用Adobe Web SDK (Alloy.js)轉譯選件的時間和位置，而不需依賴預先定義的範本或視覺版面工具。
