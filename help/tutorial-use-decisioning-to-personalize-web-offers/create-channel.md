# 建立程式碼型體驗管道

Adobe Journey Optimizer (AJO) [!UICONTROL Decisioning]中的程式碼式體驗是一種設定，可讓您使用使用者端JavaScript直接將個人化優惠傳送至網頁。 此方法可讓開發人員完全控制使用Adobe Web SDK (`Alloy.js`)呈現選件的時間和位置，而不需依賴預先定義的範本或視覺版面工具。
