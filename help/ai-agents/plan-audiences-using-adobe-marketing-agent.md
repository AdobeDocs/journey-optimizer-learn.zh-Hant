# 使用 Adobe Marketing Agent for Microsoft 365 Copilot 規劃客群

瞭解 Adobe Marketing Agent for Microsoft 365 Copilot 如何直接在 Microsoft Teams 中簡化客群規劃。此工具可讓行銷人員識別忠誠度客群、檢查歷程譜系及驗證行銷活動候選者，而無需切換平台或向營運團隊提出請求。

此影片示範選取、分析和準備行銷活動客群的流程。它會展示代理程式如何產生可操作的深入解析、顯示相關建議，並提供客群與歷程之間的譜系視覺效果。行銷人員可以有效地管理資料並簡化工作流程，確保目標定位更聚焦且行銷活動結果更佳。
