# 使用AI建立品牌內電子郵件內容

瞭解Adobe Journey Optimizer中的AI功能如何協助您大規模建置品牌內電子郵件內容。 此逐步解說會顯示從單一提示產生的完整電子郵件，使用品牌指引和參考內容來引導產生、選取產生模型，以及將AI產生的文字和影像直接插入範本。 相同的品牌資產也能支援自動化品質檢查，針對每個變體評分是否符合規範、音調和品牌一致性。
