# 為行銷人員在Adobe Journey Optimizer中建立AI支援的品牌內內容

瞭解如何使用Adobe Journey Optimizer的AI功能來建立、調整及控管品牌上內容，從品牌設定、完整行銷活動產生，到視覺化增強功能、多語言版本和自動化品質檢查。
專為想要在各個管道擴充個人化、一致內容的行銷人員所打造。

## 您能學到的內容

* 建置並設定品牌描述檔，以引導Adobe Journey Optimizer、GenStudio和Adobe Express中的AI產生內容。
* 使用AI產生和編輯全通路電子郵件、推播和應用程式內內容。
* 使用整合式AI工具增強視覺效果並建立多語言版本。
* 在內容上線之前，執行自動化品質檢查、評估品牌一致性並套用治理。

### 簡介和品牌設定

此單元會逐步引導您在Adobe Journey Optimizer中設定品牌，讓您的AI產生的內容可反映您從一開始是誰。 您會看到AI產生電子郵件內容的完整範例，然後透過上傳指引和對應基調、法律限制和影像規則等關鍵元素來建立和設定品牌設定檔。 您也會瞭解如何將該品牌擴展到GenStudio和Adobe Express中，以及針對不同的團隊管理多個品牌。 到最後，您擁有可運作的品牌設定檔，無論您建立內容，都會有意義地引導您的AI產生內容。

* [使用AI建立品牌內電子郵件內容](./introduction.md)
瞭解Adobe Journey Optimizer中的AI功能如何協助您大規模建立品牌內電子郵件內容，從單一提示到完全產生的品牌內電子郵件。
* [在Adobe Journey Optimizer中為AI支援的內容設定品牌](./configure-brands.md)
建立並設定AI功能用來產生和驗證內容、對應語音基調、法律限制、影像規則等的品牌。
* [跨創意工具擴充品牌指導方針](./brands-tips-and-tricks.md)
瞭解如何將Adobe Journey Optimizer中設定的品牌套用至GenStudio和Adobe Express、新增範例，以及管理多個品牌。

### 使用AI產生內容

本單元說明如何在Adobe Journey Optimizer中使用AI功能端對端產生和調整品牌上內容。 您可以從提示產生完整的電子郵件、推播和應用程式內資產，編輯個別區段（例如主旨行、預覽文字和影像）、增強視覺效果和建立多語言版本，同時保留您的品牌聲音。 到最後，您會從構想變成精良、多語言、品牌上的文案和視覺效果。

* [從頭開始建立全通路內容](./create-content-from-scratch.md)
從範本開始，使用AI功能、品牌指南和可設定的色調和影像設定來產生完整電子郵件、推播通知或應用程式內訊息。
* [編輯並個人化區段內容](./edit-and-personalize-section-content.md)
使用AI功能編輯並個人化電子郵件範本的個別區段，變更主旨列、標題、文字區塊和影像，同時保留品牌聲音和合規性。
* [使用AI增強您的視覺內容](./enhance-your-visual-content.md)
使用AI工具產生資訊圖形、新增覆蓋圖並調整影像，然後在Adobe Express中擴充編輯專案，以產生隨品牌推出的視覺效果供重複使用。
* [建立並調整內容的多語言版本](./create-and-refine-multilingual-content.md)
使用AI功能、品牌指引和自動化評分將內容翻譯並調整成其他語言，讓本地化內容保持在品牌上。

### 品質、品牌認知與控管

此模組可協助您在AI產生的內容上線之前評估並控管該內容。 您可使用品牌一致性分數來根據您的指引檢查內容、執行可讀性和合規性的自動化品質檢查，並在核准程式中套用人為稽核。 到最後，您會知道如何「信任但驗證」AI輸出，並保持每個資產合規且一致。

* [確保內容品質、品牌一致性和控管](./quality-brand-alignment-and-governance.md)
評估AI產生和手動編寫資產的內容品質和品牌一致性，並在內容上線前將治理套用為檢閱程式的一部分。
