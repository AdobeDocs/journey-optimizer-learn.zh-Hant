# 在Journey Optimizer中建立品牌上內容 — 教學課程

瞭解如何在Adobe Journey Optimizer中使用AI，在整個客戶歷程中建立、調整及擴展品牌上內容。

本課程會引導您完成從建立和設定品牌指引，到產生完整行銷活動、編輯個別內容元件、增強視覺效果及啟用多語言體驗等所有工作。您也會瞭解如何套用自動化品質檢查、確保品牌一致性，以及維護所有通訊的控管。 到最後，您將能簡化內容建立、改善一致性，並跨管道提供高品質、個人化的體驗。

## 您將瞭解的內容

### 簡介和品牌設定

* [簡介](./introduction.md)
此簡介會逐步解說由AI支援的工具如何協助您產生電子郵件內容並套用品牌指引。
* [設定品牌](./configure-brands.md)
瞭解如何在Adobe Journey Optimizer中上傳品牌指引、定義樣式，以及確保跨管道一致的品牌上通訊。
* [品牌准則提示](./brands-tips-and-tricks.md)
瞭解在Adobe Journey Optimizer、GenStudio和Adobe Express之間擴充品牌指引的秘訣，以提升品牌一致性和團隊協同合作。

### 產生和調整內容

* [從頭開始建立內容](./create-content-from-scratch.md)
從範本開始，套用品牌指引，並使用簡單提示從頭開始建立電子郵件、推播通知和應用程式內訊息。
* [編輯並個人化區段內容](./edit-and-personalize-section-content.md)
瞭解如何使用AI在Adobe Journey Optimizer中編輯及個人化個別內容元件。
* [增強您的視覺內容](./enhance-your-visual-content.md)
瞭解如何在Adobe Journey Optimizer中增強視覺內容。
* [建立並調整內容的多語言版本](./create-and-refine-multilingual-content.md)
探索AI如何協助翻譯內容並將內容當地語系化，同時維持品牌一致性。 瞭解提升全域通訊規模的有效工作流程。

### 確保品質、品牌一致性和控管

* [確保品質、品牌一致性和治理](./quality-brand-alignment-and-governance.md)
瞭解如何使用AI支援的工具來評估及改善內容品質，同時維持強大的品牌一致性。
