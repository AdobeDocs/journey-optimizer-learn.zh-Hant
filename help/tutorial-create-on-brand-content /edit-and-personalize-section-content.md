# 編輯和個人化區段內容

使用Adobe Journey Optimizer中的AI功能編輯及個人化電子郵件範本的個別區段。 變更主旨列、標題、文字區塊和影像，同時保留品牌顏色、聲音和法規遵循規則。 使用提示和參考內容來調整色調、重述文字、翻譯語言，並產生每個元件的品牌視覺效果。
