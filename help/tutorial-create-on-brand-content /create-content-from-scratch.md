# 從頭開始建立全通路內容

在此步驟中，您會從現有的內容範本開始，使用AI功能產生完整的資產，例如電子郵件、推播通知或應用程式內訊息。 您可以套用品牌指南並設定語言、色調和影像設定，讓內容保持在品牌上。 您可以產生多個AI驅動型變體、檢閱品牌得分，並手動編輯內容，或使用第二個AI傳遞來調整訊息和視覺效果。
