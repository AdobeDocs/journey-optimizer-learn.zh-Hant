# 在Adobe Journey Optimizer中設定AI支援內容的品牌

您將瞭解如何建立和設定AI功能用來在Adobe Journey Optimizer中產生和驗證內容的品牌。 您可以上傳品牌指引檔案，並對應關鍵元素，例如語調、口號、法律限制、影像規則和CTA格式。 您可以編輯管道和元素層級的指引，以改進產生精確度及自動化品質檢查。
