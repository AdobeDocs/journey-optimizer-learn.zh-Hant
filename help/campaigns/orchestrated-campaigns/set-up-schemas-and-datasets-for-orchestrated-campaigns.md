# 設定行銷活動的結構描述和資料集

瞭解如何在 Adobe Journey Optimizer 中為協調的行銷活動設定關聯式結構描述和資料集。本指南說明如何定義索引鍵、關聯，以及如何使用 DDL 檔案來有效建立結構描述。您也會瞭解如何透過 CSV 上傳和變更資料擷取 (CDC) 來擷取資料、驗證資料集，以及準備查詢、個人化和行銷活動執行。觀看內嵌影片，瞭解簡化行銷活動工作流程的逐步指示。
