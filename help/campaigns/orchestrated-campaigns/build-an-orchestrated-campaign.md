# 建立協調的行銷活動

瞭解如何透過建立結合客群、資料操控和多管道傳送的工作流程，在 Adobe Journey Optimizer 中設計和執行協調的行銷活動。此影片逐步解說行銷活動排程、從關聯式資料建立客群、擴充和重複資料刪除，以及使用多個串流支援進階使用案例，例如捨棄購物車後續追蹤、防止流失和補貨通知。
