# 在行銷活動中利用訊息最佳化

了解如何在動作中或 API 觸發的行銷活動中運用訊息最佳化。您可以瞭解如何鎖定子對象、依位置建立訊息變化、啟用遞補內容，以及在單一行銷活動中執行多個實驗。 本教學課程也涵蓋如何管理多頻道行銷活動，同時維持訊息的一致性。
