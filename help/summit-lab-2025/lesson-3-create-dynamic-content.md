# 第3課 — 建立動態內容

*您的挑戰：*&#x200B;您必須確定僅要求尚未安裝應用程式的客戶進行安裝。 已安裝應用程式的任何人應該只會收到一般歡迎訊息。 您不想讓歷程流程更複雜。 

*解決方案*：動態內容可讓您根據可由設定檔屬性、情境式事件或對象組成的條件規則，調整訊息的內容。 

在此練習中，您以套用動態內容至SMS為例，但可將其套用至所有管道。
