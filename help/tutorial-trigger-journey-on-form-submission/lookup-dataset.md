# 建立查詢資料集

資料集查詢可讓Adobe Journey Optimizer在歷程執行階段從Adobe Experience Platform資料集擷取參考或交易式資料，而不需要將該資料儲存在即時客戶設定檔上。

在此使用案例中，當使用者提交投資偏好設定表單（股票、債券或CD）時，歷程會從參考資料集中動態查詢對應的投資詳細資料，並使用該資訊來個人化電子郵件。
