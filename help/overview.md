# Journey OptimizerTutorials — 概觀

![](./assets/ajo-banner.png)

Journey Optimizer可協助公司為客戶提供連結、情境式和個人化的體驗。 客戶歷程是從第一次接觸直至客戶離開，客戶與品牌互動的整個過程。 

本使用手冊包含有關AdobeJourney Optimizer功能的影片和教學課程。
