# 報告和監視

Adobe Journey Optimizer的報告功能是將原始資料轉換為可操作的深入分析，協助您調整客戶歷程和行銷策略。 瞭解如何存取概述量度、深入瞭解行銷活動和歷程特定報告，以及透過Customer Journey Analytics解鎖進階分析。

無論您是初學者或經驗豐富的行銷人員，這些工具都能提供清晰明瞭的品質，協助您最佳化達成有意義的商業成果。
