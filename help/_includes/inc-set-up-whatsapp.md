# 設定 WhatsApp 頻道

本教學課程將引導您在 Adobe Journey Optimizer 中設定 WhatsApp 頻道，以實現即時業務訊息傳送。

您將學到：

* 新增並設定 WhatsApp API 憑證和企業帳戶詳細資訊。
* 設定用於傳入訊息處理的 Webhook，包括選擇加入/選擇退出關鍵字。
* 建立頻道設定以完成您的 WhatsApp 設定。

非常適合希望在客戶歷程中啟用 WhatsApp 訊息的行銷人員和開發人員。
