# 在行銷活動中運用訊息最佳化

瞭解如何在行動中或API觸發的行銷活動中運用訊息最佳化。 您將瞭解如何鎖定子對象、建立依位置的訊息變化、啟用遞補內容，以及在單一行銷活動中執行多個實驗。 本教學課程也涵蓋如何管理多管道行銷活動，同時維持訊息的一致性。
